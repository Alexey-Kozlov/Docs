Создаем проект с использованием xUnit

Создаем класс, нужный метод помечаем - [Fact]
На закладке слева - где колба - вверху нажать "Обновить" - для пересканирования проектов и нахождения тестовых методов.
Должны появится тетовые методы.

Чтобы работал вывод в консоль.

Для этого открываем панель Output (Ctrl K  Ctrl H), в правой части от раздела с панелями - в выпадающем списке выбираем C# DevKit - Test Explorer. Теперь в этом окне будет отображаться все что вывели командой Console.WriteLine()
