должен быть установлен Prettier

включаем автоформатирование при сохранении
- File - Preferences - Settings,
в строке поиска указываем - formatter
выбираем везде форматтер Prettier, также ставим галки форматировать при сохранении и вставке

Если не работает форматирование файлов TypeScript -

- открываем файл, выделяем весь текст, жмем Ctrl-Shift + I
- будет всплывающее окно - что не установлен форматтер для файлов TypeScript, выбираем из предложенных вариантов Prettier
- после этого должно корректно работать форматирование
- в правом нижнем углу редактора в строке статуса должна появиться надпись - <двойная галка>Prettier, т.е. используется Prettier для работы с TypeScript
