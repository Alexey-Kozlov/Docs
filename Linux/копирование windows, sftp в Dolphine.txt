Возможно подключиться к другому компу через WiFi не только через Krusaider, но также и через обучный проводник Dolphine.
Для этого в строку пути к папке указываем:
sftp://<имя пользователя удаленного компа>@<IP удаленного компа>:22/<Диск>:
Далее можно указать путь к нужной папке, но не обязательно.
Жмем Enter, может спросить пароль, указываем если спросила, далее открывается удаленный комп .

Пример подключения:

sftp://alex@192.168.1.5:22/D:/Torrent/
sftp://alex@192.168.1.5:22/E:


Копирование из винды в Linux

1. Проверяем - работает ли ssh:
ssh
должен появится хелп

Коннектимся из винды к линуксу:
ssh alex@192.168.1.11
Могут спросить - установить ли ключи соединения, пишем - yes
должен быть спрошен пароль, указываем - попажаем на Линуксовый комп

exit - выход из линуксового компа



2. Копируем утилитой scp через командную строку, из винды в линукс
на винде пишем:
scp -r C:\Users\user\Downloads\*.* alex@192.168.1.11:/home/alex/Temp
Будет спрошен пароль, потом пойдет копирование
флаг -r - это копирование с подкаталогами



3. Можно копировать с помощью утилиты sftp
проверяем - установлена ли sftp на винде - sftp

Далее подключаемся к линуксу:
sftp alex@192.168.1.11
должен быть спрошен пароль, указываем - попажаем на Линуксовый комп

далее копируем с линукса в винду:
sftp> get /home/alex/Temp/*.*

Будет копирование, для выхода из утилиты - Ctrl-C

скопированный файл ищем в винде - C:\Users\User, т.е. в каталоге текущего пользователя.

копируем из винды в линукс:
sftp> put C:\Users\User\downloads\*.*
