Если при установки какого-либо софта (например через flatpak) будет ошибка доступа - 403 - forbidden из-за санкций,
делаем так.

1. скачиваем файл дистриба h264 - https://web.archive.org/web/20251005202247/http://ciscobinary.openh264.org/libopenh264-2.5.1-linux64.7.so.bz2

2. Правим - /etc/hosts, добавить -  127.0.0.1 ciscobinary.openh264.org

3. cd Downloads (или cd Загрузки)

4. Запускаем локальный веб-сервер - sudo python3 -m http.server 80, не закрываем это окно терминала!

5. Запускаем установку через flatpak, должно теперь подтянуться эта библиотека (libopenh264-2.5.1-linux64.7.so.bz2) из локального вебсервера. Если это произошло - в открытом окне терминала с запущенным локальным веб-сервером будет сообщение, чтото типа - 127.0.0.1 - - [15/Feb/2026 14:10:02] "GET /libopenh264-2.5.1-linux64.7.so.bz2 HTTP/1.1" 200 - значит, все прошло штатно.

6. Такую технику можно использовать для любой другой инсталяции - когда обращение идет к внешнему серверу, но нужно перенаправить на локальный с заранее подготовленным файлом.
