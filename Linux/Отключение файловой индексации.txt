Время от времени самопроизвольно запускается файловая индексация.
Она жрет проц и грузит дисковую систему.
Для отключения вводим команду:
sudo balooctl6 suspend
sudo balooctl6 disable
sudo balooctl6 purge
Можно перед этим через taskManager удалить процесс baloo_file

Если не помогает - удалить из /usr/bin все файлы, начинающиеся на baloo*

Индексы, которые делает этот baloo, находятся в ./local/share/baloo  Эту папку можно удалять, если есть
