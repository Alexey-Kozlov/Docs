Установка

утилита iwctl - настройка wifi
device list
station wlan0 scan
station wlan0 get-networks
station wlan0 connect DIR-X1530-5G-ADD5
exit

делаем разделы
просмотр разделов - lsblk (или lsblk -f   - подробно)
cfdisk /dev/nvme1n1
cfdisk /dev/nvme0n1
правим разделы этой утилитой, список разделов:
/dev/nvme1n1p1  - EFI
/dev/nvme1n1p2  - SWAP
/dev/nvme1n1p3  - основной раздел
/dev/nvme0n1p1  - дополнительный раздел на втором физическом диске

форматируем:
mkfs.ext4 /dev/nvme1n1p3  -это основной раздел
mkfs.ext4 /dev/nvme0n1p1  -это дополнительный раздел
mkfs.fat -F32 /dev/nvme1n1p1  - efi
mkswap /dev/nvme1n1p2  - swap форматируем
swapon /dev/nvme1n1p2  - swap включаем

монтируем:
mount /dev/nvme1n1p3 /mnt  - монтируем корневой раздел
создаем нужные точки монтирования:
mkdir /mnt/efi
mkdir /mnt/home
mkdir /mnt/files
монтируем нужные разделы на созданные точки:
mount /dev/nvme1n1p1 /mnt/efi
mount /dev/nvme1n1p3 /mnt/home
mount /dev/nvme0n1p1 /mnt/files

устанавливаем основные компоненты:
pacstrap /mnt base linux linux-firmware

делаем fstab:
genfstab -U /mnt >> /mnt/etc/fstab

подключаемся под рутом:
arch-chroot /mnt /bin/bash

устанавливаем редактор vim:
pacman -S vim

настраиваем синхронизациювремени:
exit - выходим  из рута
timedatectl set-ntp true
timedatectl set-timezone Europe/Moscow 
timedatectl status
ln -sf /usr/share/zoneinfo/Europe/Moscow /etc/localtime
hwclock --systohc

локализация
arch-chroot /mnt /bin/bash
vim /etc/locale.gen
раскомментируем строки - en_US.UTF-8 UTF-8   ru_RU.UTF-8 UTF-8
сохраняем: <Esc> :w <Enter> :q
locale-gen
ставим ИЛИ английскую локаль - echo "LANG=en_US.UTF-8" > /etc/locale.conf
ставим ИЛИ русскую локаль - echo "LANG=ru_RU.UTF-8" > /etc/locale.conf
добавляем русский шрифт в локаль:
echo "FONT=cyr-sun16" > /etc/vconsole.conf

наименование компа
vim /etc/hostname
<Insert>
указываем в первой строке наименование компа- arc
<Esc>
:w <Enter> :q <Enter>

правим hosts:

vim /etc/hosts
<Insert>
127.0.0.1  localhost
::1        localhost
127.0.0.1  arc.localdomain arc
<Esc>
:w  <Enter> :q

ставим пароль на root:
passwd

Делаем  нового пользователя:
useradd -m alex
passwd alex
usermod -aG wheel,audio,video,storage alex

ставим sudo:
pacman -S sudo

редактор по умолчанию:
EDITOR=vim

делаем расширенные права у обычного пользователя:
visudo
раскомментировать - %wheel ALL=(ALL) ALL

Ставим  загрузчик:
pacman -S grub efibootmgr

-------------------------------------
если ставим только Linux:

grub-install /dev/nvme1n1 --efi-dir=/efi
пересобираем загрузчик
mkinitcpio -p linux
обновляем конфиг
grub-mkconfig -o /boot/grub/grub.cfg

--------------------------------------
если ставим Linux после винды - для двойной загрузки:

ставим os-prober:
pacman -S os-prober
правим загрузчик Grub:
vim /etc/default/grub
в открытом файле раскомментировать последнюю строку:
GRUB_DISABLE_OS_PROBER=false
сохранить

Перегружаем!

Проверяем разделы:
lsblk -f
должна быть структура:
первые 3 раздела - винда:
- nvme0n1p1 - раздел efi - FAT32
- nvme0n1p2 - служебный раздел
- nvme0n1p3 - основной раздел - ntfs
дальше 3 раздела Linux:
- nvme0n1p4 - раздел efi - FAT32
- nvme0n1p5 - раздел с файлом подкачки - SWAP
- nvme0n1p6 - основной раздел - ext4

Монтируем:
mkdir /mnt/boot
mount /dev/nvme0n1p6 /mnt
mount /dev/nvme0n1p4 /mnt/boot

заходим в root:
arch-chroot /mnt
проверяем - утилита os-prober должна найти винду:
os-prober
длжен быть вывод - что найден загрузчик винды
компилируем и правим загрузчик:
mkinitcpio -p linux
обновляем конфиг
grub-mkconfig -o /boot/grub/grub.cfg
смотрим на вывод - сначала должен быть вывод что найден linux, несколько строк
затем - что запущена утилита os-prober и найдена винда

Проверяем, что получилось в Grub:
vim /boot/grub/grub.cfg
там в конце скрипта должен быть раздел о загрузке обычного и продвинутого Линукса и о загрузке винды.
