По умолчанию среда выполнения .Net передает данные телеметрии.

Для отключения телеметрии (для текущего пользователя):
- открыть на редактирование файл - ~/.bashrc
- добавить в самый низ строку:
export DOTNET_CLI_TELEMETRY_OPTOUT=1
сохраняем, переоткрыть сессии

либо для отключения телеметрии (для всех пользователей):

добавить в файл /etc/environment строку:
DOTNET_CLI_TELEMETRY_OPTOUT=1

Проверить установленные переменные можно командой:
printenv  - отобразятся все переменные
printenv | grep DOTNET  - только относящиеся к DotNet
