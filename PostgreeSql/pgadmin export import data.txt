Для экспорта и импорта в PgAdmin4 путь по умолчанию следующий
- контейнер где запущен PgAdmin
путь :
var -> lib -> pgadmin -> storage -> admin_admin.com
эту папку открывает UI PgAdmin4 при экспорте и импорте

Все данные можно экспортировать напрямую через UI контейнера.

Также - эта папка автоматически замаплена в докере,
открываем в контейнере PgAdmin4 рахдел Inspect, в нем - раздел Mounts, в нем - название смонтированного тома, это параметр Name
Это служебный маппинг, поэтому его можно найти в разделе докера Volumes, ищем замапленный том по его наименованию Name (из раздела Inspect)
Открываем том, там та же файловая система, копируем оттуда что нам нужно
